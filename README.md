# Computer Networking Project

This project demonstrates the design and configuration of a computer network using Cisco Packet Tracer.

## Features

- Designed a network topology
- Assigned IPv4 addresses
- Configured routers and switches
- Tested connectivity using ping
- Applied basic routing and network security concepts

## Tools Used

- Cisco Packet Tracer
- IPv4 addressing
- Routers and switches
